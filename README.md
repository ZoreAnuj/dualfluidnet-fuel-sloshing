# DualFluidNet: Attention-based Dual-pipeline Network for Fluid Simulation

This project implements a neural network method for efficient and robust fuel sloshing simulation in aircraft. It explores using an attention-based dual-pipeline architecture to model complex fluid dynamics, aiming to provide faster simulations compared to traditional computational fluid dynamics (CFD) methods.

## Key Features
*   Attention-based dual-pipeline network architecture for fluid state prediction.
*   Specialized for simulating fuel sloshing dynamics in aerospace applications.
*   Focus on efficiency and robustness for complex moving boundary problems.

## Tech Stack
*   Python
*   PyTorch
*   NumPy

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/dualfluidnet-fuel-sloshing.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Follow the instructions in `train.py` to begin training the model.
